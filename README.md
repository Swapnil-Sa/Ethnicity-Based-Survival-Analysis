# 🧠 Breast Cancer Survival Analysis Using Python

A Python-based data analysis project focused on exploring breast cancer survival patterns using real-world data. This project applies data cleaning, exploratory analysis, feature evaluation, and statistical techniques to identify key factors influencing survival outcomes.

---

## 📌 Project Overview

This project performs end-to-end data analysis on breast cancer survival data. The goal is to understand how different variables affect survival outcomes and to extract meaningful insights using Python.

The analysis includes:
- Data preprocessing and cleaning
- Exploratory Data Analysis (EDA)
- Feature importance analysis
- Survival comparison across groups
- Visualization of key findings

---

## 🎯 Objectives

- Clean and preprocess raw dataset for analysis  
- Explore survival trends using visualizations  
- Identify important clinical and demographic features  
- Analyze relationships between variables and survival outcome  
- Apply statistical techniques to extract insights  
- Present results in a clear and interpretable manner  

---

## 🛠️ Tools & Technologies

- Python  
- Pandas  
- NumPy  
- Matplotlib  
- Seaborn  
- Scikit-learn  
- Jupyter Notebook  

---

## 🔄 Workflow

1. Data Loading & Inspection  
2. Data Cleaning & Preprocessing  
3. Exploratory Data Analysis (EDA)  
4. Correlation Analysis  
5. Mutual Information Analysis  
6. Stability-based Feature Selection  
7. Survival Comparison & Visualization  
8. Interpretation of Results  

---

## 📊 Key Results & Insights

- Survival outcomes are not uniform and vary across different groups in the dataset  
- Disease severity indicators such as tumor size and stage show strong influence on survival  
- Feature analysis techniques (correlation, mutual information, stability selection) consistently identified key variables affecting outcomes  
- Survival probability decreases as disease severity increases  
- Visualizations clearly highlight patterns and differences in survival trends  
- Statistical analysis confirms that some variables have significantly higher impact on survival compared to others
